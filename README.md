# 📊 Telco Customer Churn Analysis

## Project Overview
Analysis of customer churn for a telecommunications company 
using Excel 2019. The project identifies key drivers of churn 
and provides actionable recommendations for management.

**Dataset:** 7,043 customers | 21 variables
**Tool:** Microsoft Excel 2019
**Key Metric:** Overall Churn Rate — 26.5%

---

## 🚨 Key Findings

| Segment | Category | Churn Rate |
|---|---|---|
| Contract Type | Month-to-Month | 42.71% |
| Internet Service | Fiber Optic | 41.89% |
| Payment Method | Electronic Check | 45.29% |
| Senior Citizens | Senior | 41.70% |
| Tenure Group | New Customers (0-12 months) | 47.44% |

---

## 💡 Business Recommendations

1. **Convert month-to-month customers** to annual contracts 
through incentives and discounts — churn drops from 42.71% 
to 2.83% for two year contracts
2. **Investigate Fiber Optic service quality** — premium 
customers churning at 41.89% suggests a value for money problem
3. **Enroll customers in automatic payments** — electronic 
check customers churn at 45.29% vs 16% for automatic payments
4. **Focus retention efforts on new customers** — 47.44% churn 
in first year means onboarding experience needs improvement
5. **Create senior citizen retention program** — seniors churn 
at nearly double the rate of non-seniors
